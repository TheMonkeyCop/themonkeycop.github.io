This can be due to windows or incorrect device being used.


    Try the following:
  1. Go to Start -> Select settings ->Then go to system ->Sound ->Sound Control Panel
  2. On there select the "Recording" Tab
  3. Make sure your mic is default device.
  4. Then go to properties ->"Advanced" Tab ->Uncheck "Allow applications to take exclusive control of this device"

When In-game...
  1. Esc ->Settings ->Voice Chat
  2. Make sure your mic is selected for Input Device.
  3. With this menu open, your voice chat will playback to you, but not to the server.
  4. Toggle mic off and on (Note: You may need to do this step specifically every time you join the server)
