Keybinds

  • M - interaction menu (use enter, arrows, and backspace to navigate)
  • X - hands up
  • X + alt - hands behind head
  • Z - change voip range
  • Run + e - tackle
  • Run + ctrl - dive to the ground
  • Tap ctrl - cycle through stances (attack stance, crouch, prone)
  • DEL - drop weapon currently in hand

Keybinds

  • Arrow Up - bring up the phone
  • END - exit out of text input
  • ENTER - select / select input box
  • BACKSPACE - navigate backwards
  • RIGHT CTRL - full screen mode
  • CTRL + S - save contact
  • Arrow Left - on messages screen and on a contact - delete a sequence of messages *WILL DELETE ALL MESSAGES
  • Arrow Right - deselect for deletion as done above

  • w/ microphone equipped:
    • - Alt + E to toggle microphone up to player face on air
    • - Alt + 1 to toggle interview on air
  • w/ camera equipped:
    • - Alt + E to toggle camera on shoulder
    • - Alt + 1 to toggle live on air

Trucking

  • - /trucking - Base Command - run this to get a prompt
  • - /trucking return - Force return the truck. Use this when shit hits the fan and you lost it or it's in the ocean.
  • - /trucking reset - Force reset the job
  • - /trucking time - Show time remaining on the job (if you're on one)
  • - /trucking tacho - Show tachograph to players nearby (works like /id)
    • Accepted variations:
      • showtacho
      • tachograph
      • showtachograph
